# PRD Authoring Skill

## Overview
Turn the user request into a structured PRD. The PRD is the *what*; the
architecture skill is the *how*.

## Capabilities
- Define user personas
- Enumerate functional and non-functional requirements
- Apply MoSCoW priority
- Write measurable acceptance criteria
